什么是java bean

  1. bean是对象；

  2. 由Spring中一个叫loC的东西管理；

  3. 我们的应用程序由一个一个bean构成；

Bean的作用域

Bean在Spring中的某种行为模式：

Bean对象是默认的单例作用域，如果有任何地方有任何修改了这个单例的对象，那么其他类再调用就会得到一个修改后的值；

作用域分类

  1. singleton：单例作用域；

  2. prototype：原型作用域；

  3. request：请求作用域；

  4. session： 会话作用域；

  5. application：全局作用域；

后三种作用域，只适用于Spring MVC框架；

bean规范

  1. 所有属性为private；

  2. 提供默认构造方法；

  3. 提供setter和getter；

  4. 实现serializable接口；
